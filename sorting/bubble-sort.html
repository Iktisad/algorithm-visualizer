Bubble Sort

Pseudocode

for i = 0 to n-2:
for j = 0 to n-2-i:
if array[j] > array[j+1]:
swap(array[j], array[j+1])

Current line highlighted shows which part of the algorithm is executing.

The outer loop controls passes through the array, while the inner loop compares adjacent elements.

Bubble Sort Visualization

How Bubble Sort Works

Bubble Sort repeatedly compares adjacent elements and swaps them if they are in the wrong order. The largest element "bubbles up" to its correct position after each pass. This continues until the array is fully sorted. Watch the yellow bars being compared, red bars being swapped, and green bars showing sorted elements.
